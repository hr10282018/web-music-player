### web-music-player（网页版的音乐播放器）
做这个网页版的音乐播放器，本意是当作小 demo，为了练习和巩固 CSS、 JS。

完成的功能应该没有什么bug，但是因时间问题，还有部分的功能和优化还没做，一些浏览器兼容没有考虑。

欢迎大佬小佬们给建议！！
